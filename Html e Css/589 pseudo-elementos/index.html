Pseudo-Estados


Vamos praticar sobre pseudo-elementos:

Pseudo-estado é diferente de pseudo-classes. Pseudo-classes é quando uma classe está representando um estado de forma não explicita. Por exemplo, um link já vem por seu padrão um estado de "Link", como um input radio já vem por seu padrão um estado não explicito de unchecked


Pseudo-estados se refere a criação pseuda de um estado. vou criar um elemento onde ele só existe no css. Ele é tratado com a representação (::) no CSS. pode ser usado por exemplo:
ul li::after { contente : "Criando algo"; }
